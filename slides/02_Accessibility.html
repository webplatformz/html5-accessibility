Standard

  • WAI = Web Accessibility Initiative (~ W3C)
  • WCAG = Web Content Accessibility Guidelines
There is no Standard - just guidelines !

WCAG

  • Multiple versions
  • Incremental enhancements & adjustments
  • Depending on the (X)HTML version in use
Not mandatory, not binding, no guarantee for success !

(General) Best Practice

  • Start small
  • Keep it simple
  • Be minimalistic
  • Go for meaningful semantic HTML
  • Provide "Fallbacks"

Personal Preference

  • Strict (X)HTML 5
  • Strict Javascript only
  • Strict Css only

Pro & Cons of Strictness

  • No empty tags
  • No empty attributes
  • No non-closing tags
  • No invalid html, css, js
  • Strict quoting
  • Strict encoding
  • More effort
  • Incompatibility
Maximum validity, predictability, security
